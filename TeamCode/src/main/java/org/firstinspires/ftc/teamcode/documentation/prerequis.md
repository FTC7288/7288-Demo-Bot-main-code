- installation de git + config + powershell (lire git.md)

- rajouter sa clé sur le repo robolyon (si possible)

# rappel commandes linux

- aide: sur chaque commande on peut rajouter `<command> --help`
- repertoire courant: `pwd` (print working directory)
- liste le contenu du repertoire courant: `ls` ou `ls -l` ou `ls -al`
- creer un repertoire : `mkdir <name>` -> `mkdir helloworld`
